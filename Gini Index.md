# Gini Index

* `G(S) = 1 * (c∈C)∑[p(c)--2]`
    * `G(S)` : Gini index of set  `S`.
    * `C` : the set of classes in `S`, e.g., for binary classification, `C={0, 1}`.
    * `p(c)` the proportion of the number of elements in class `c` to the number of elements in set  `S`.
* Favors larger partitions.
